## Project Cards
- `[[projects]]`: cross-project hub with short descriptions and links into each project subtree.
- `[[project-overview]]`: current project scope and durable context.
- `[[architecture]]`: structure, components, and technical decisions.
- `[[commands]]`: verified commands for running and checking the project.
- `[[tasks]]`: durable follow-ups and roadmap items.
- `[[session-summaries]]`: short summaries of important AI sessions.
- `[[known-issues]]`: known problems, limitations, and workarounds.

## Notes
- Use this as the central page for the memory graph, not as a replacement for the index.
